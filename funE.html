Função Exponencial

O que é uma função exponencial?

Uma função exponencial é uma função que tem uma incógnita elevada a uma variável. A incógnita normalmente é nos dada no enunciado e a chamammos de a. A variável é chamada de x.

Gráfico

O gráfico da função exponencial é formado por uma curva exponencial que cresce ou decresce muito rapidamente.

Nesse tipo de função temos apenas a presença do parametro a, que indica se a curva exponencial é crescente (a > 0) ou decrescente (0 < a < 1). a nunca encosta no eixo x.

Construindo um gráfico

Para construir um gráfico de uma função exponencial, basta atribuir valores para x, e descobrir seu f(x). Fazendo isso você consegue as cordenadas x e y, aí é só aplicar no gráfico.

Raiz de uma função exponencial

A raiz de uma função exponencial, quando não há valor de b, é sempre 1 pois qualquer numero elevado a zero é um.
Mas, caso surja a duvida, eleve a base a 0 para achar a raiz.
